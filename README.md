## Usage

1. Pick one way to install the addon.

    ```bash
    $ yarn add eslint-config-konrad --dev
    $ npm install eslint-config-konrad --save-dev
    ```

2. Inherit in your eslint config (`.eslintrc` or `.eslintrc.js`)
    ```json
     {
      "extends": "eslint-config-konrad",
      "rules": {}
     }
    ```
